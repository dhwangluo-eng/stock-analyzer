# Stock Analyzer - 股票分析助手

基于7维度预测体系的股票技术分析工具。

## 功能

- **实时行情**：获取最新价格、涨跌幅
- **技术分析**：MA均线、RSI、KDJ、布林带
- **资金流向**：量比、换手率分析
- **综合评分**：7维度评分体系
- **预测建议**：买入/持有/减仓建议

## 使用方法

### 命令行
```bash
# 分析单只股票
python3 stock_analyzer.py 600406

# 分析多只股票
python3 stock_analyzer.py 600406
python3 stock_analyzer.py 000001
```

### 作为模块导入
```python
from stock_analyzer import analyze_stock, format_report

result = analyze_stock("600406")
print(format_report(result))
```

## 7维度评分体系

| 维度 | 权重 | 说明 |
|:---:|:---:|:---|
| 技术面(MA) | 30% | 均线多头排列/空头排列 |
| RSI指标 | 15% | 超买超卖判断 |
| KDJ指标 | 15% | 金叉死叉信号 |
| 资金流向 | 20% | 量比、换手率 |
| 新闻舆情 | 10% | 利好利空分析 |
| 大盘环境 | 5% | 市场强弱 |
| 板块轮动 | 5% | 所属板块表现 |
